Guida Configurazione
1
Abilita 2FA Sul tuo account Google
2
Genera App Password Account Google > Sicurezza > Password per le app
3
Configura .env Aggiungi GMAIL_USER e GMAIL_APP_PASSWORD
4
Test Usa la funzione di test email
Link utili:
  • Google Account Security
  • App Passwords Help
Esempio Configurazione .env 
# Gmail Configuration
GMAIL_USER=tuo_email@gmail.com
GMAIL_APP_PASSWORD=abcd efgh ijkl mnop  # Password di 16 caratteri

# Opzionali
EMAIL_SENDER_NAME=Sistema Aste Immobiliari
EMAIL_REPLY_TO=noreply@tuodominio.com
Sicurezza
  • Non condividere mai la tua App Password
  • L'App Password è diversa dalla password del tuo account Google
  • Rimuovi gli spazi dalla App Password quando la copi nel file .env
  • Riavvia l'applicazione dopo aver modificato il file .env
